## Prueba 
---

Implementada en *Nodejs*

--- 

### Ejecución de la solución

* Instalar node v16.14.0 o superior

https://nodejs.org/en/

* Clonar el repositorio 

```sh
git clone https://github.com/calmarti/Test.git
``` 

* Instalar dependencias 
```sh
npm install
```

* Ejecutar:

```sh
node index.js
```
